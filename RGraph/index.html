Introduction to RGraph

RGraph is a Javascript charts and graph library. It uses the HTML5 canvas tag to produce a wide variety of javascript charts and graphs. RGraph is covered by the RGraph License.

Faster websites with RGraph

Javascript charts are created in the users web browser, meaning quicker charts and less server load. RGraph is an easy way of speeding up your website.

The RGraph license

RGraph is covered by the RGraph license. A summary is that for commercial/business use there is a small one-time licensing fee of 49 GBP (roughly 80 USD) to pay. For non-commercial purposes it's freely usable. There are some licensing FAQs that should help to answer any questions you might have. If you need one, you can get an invoice here.

If you have any questions about RGraph licensing, you can send your question to: licensing@rgraph.net. If your question is of a support nature though, please use the support forum.
